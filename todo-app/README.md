# 📝 To-Do List App

## 📌 Description
This is a simple To-Do List web application that helps users manage their daily tasks efficiently.

## ✨ Features
- Add tasks
- Delete tasks
- Mark tasks as complete ✔️
- Interactive UI

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
